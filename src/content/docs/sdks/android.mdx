<Aside type="note" title="Coming soon">
  The Tolka Android SDK is under development. Target availability: Q3 2026.

  [Join the early access list →](https://tolka.health/contact)
</Aside>

The Android SDK will provide:

- **`TolkaClient` for Kotlin** — coroutine-native, typed API client
- **Jetpack Compose components** — `TolkaTranslate` and `TolkaVoice` composables
- **Offline mode** — Emergency domain support with on-device inference for zero-connectivity environments (Emergency plan only)
- **Biometric auth** — Integration with Android BiometricPrompt for key storage

---

## Planned API surface

```kotlin
// Planned — not yet available

    sourceLang = "nb",
    targetLang = "en",
    domain = TolkaDomain.EMERGENCY,
```
