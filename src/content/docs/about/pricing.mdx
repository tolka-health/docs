## Plan overview

| | Community | Starter | Pro | Enterprise | Emergency |
|--|-----------|---------|-----|------------|-----------|
| **Price** | Free / ~€50/mo | €50–100/mo | €500–2,000/mo | €20K–100K/yr | €1,000–3,000/mo |
| **Target** | NGOs, refugee services | Individual GPs, small clinics | Municipal health services | Hospital systems, RHFs | Ambulance/ER services |
| **Translations/mo** | 200 | 500 | 5,000 | Unlimited | Unlimited |
| **Req/min** | 20 | 60 | 300 | Custom | 500 |
| **Back-translation** | ✓ | ✓ | ✓ | ✓ | ✓ |
| **Risk scoring** | ✓ | ✓ | ✓ | ✓ | ✓ |
| **SDK** | Web | Web | Web + Mobile | All + Custom | All + Offline |
| **Support** | Community | Email | Dedicated | 24/7 + SLA | 24/7 + SLA |
| **Deployment** | Cloud | Cloud | Cloud | Cloud or On-Prem | Cloud + Offline |

## Platform tier

For telehealth platforms integrating Tolka into their product (e.g., video consultation platforms, EHR vendors), a revenue-share Platform tier is available. Contact [sales@tolka.health](mailto:sales@tolka.health).

---

## Pilot programme

Norwegian municipalities, clinics, and hospitals can apply for a **3-month free pilot** on Pro-tier access with dedicated onboarding. Apply at [tolka.health/pilot](https://tolka.health/pilot).
